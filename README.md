# Практическая работа №9

## Описание
В работе реализован отдельный экран загрузки для Android-приложения с простым анимированным оформлением.

## Что было сделано
1. В проект добавлена новая `SplashActivity`, используемая как промежуточный стартовый экран.
2. Создан layout `splashscreen.xml` без `ActionBar`.
3. На экране размещены название приложения, изображение робота и горизонтальный `ProgressBar`.
4. Подготовлен drawable-ресурс `robot.png` для оформления splash-экрана.
5. В `MainActivity` настроен переход на экран загрузки при старте приложения.
6. Через `Handler` добавлена задержка перед открытием основного окна.
7. Для изображения подключена простая fade-in анимация из XML-ресурса.

## Результат
Проект демонстрирует, как добавить splash-экран с отдельной activity, анимацией и автоматическим переходом к основному содержимому приложения.
