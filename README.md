# Практическая работа №5
## Задание: Разработка приложения
Концепция приложения: замена стандартной иконки приложения на собственную адаптивную.

## Проделанная работа:
1. Создана собственная адаптивная иконка `my_logo`.
2. С её помощью сгенерированы адаптивные иконки.
3. В `AndroidManifest.xml` подключены `android:icon` и `android:roundIcon` к новым ресурсам.
4. Оставлен базовый экран активности для проверки установки иконки.

## Итоги
Приложение использует пользовательскую адаптивную иконку, что закрепляет работу с ресурсами `mipmap` и настройкой манифеста.
