# Практическая работа №9

## Тема
Разработка экрана загрузки (`Splash Screen`) для Android-приложения.

## Цель работы
Дополнить базовое приложение отдельным экраном загрузки, настроить переход между активностями и применить простую анимацию интерфейса.

## Исходные данные
В качестве основы использовалось пустое Android-приложение с ранее заменённой иконкой. Основной экран приложения оставлен для последующего перехода после завершения загрузки.

## Выполненная работа
1. Проведён анализ исходной структуры проекта и истории коммитов.
2. Добавлен новый экран `splashscreen.xml` без `ActionBar`.
3. На экране загрузки размещены:
   - название приложения в верхней части;
   - изображение робота в нижней части;
   - горизонтальный `ProgressBar` над изображением.
4. Изображение `robot.png` подключено через `drawable`.
5. Создана отдельная активность `SplashActivity`.
6. Реализован переход из `MainActivity` на экран загрузки при старте приложения.
7. Настроена задержка показа splash-экрана на 3 секунды, после чего открывается основной экран.
8. Добавлен ресурс анимации `fadein.xml`:
   - начальная прозрачность: `0`;
   - конечная прозрачность: `1`;
   - длительность: `4000 мс`.
9. Анимация применена к изображению робота.
10. Для визуального эффекта загрузки настроен горизонтальный `indeterminate` progress bar.

## Результат
В приложении реализован отдельный экран загрузки с анимированным изображением и индикатором процесса. Работа закрепляет навыки:
- создания новых `Activity`;
- работы с `layout`-ресурсами;
- подключения изображений в `drawable`;
- использования XML-анимаций;
- настройки навигации между экранами Android-приложения.

## Вывод
В ходе работы пустое приложение было расширено экраном загрузки с анимацией и визуальным индикатором ожидания. Это позволило отработать базовые приёмы оформления стартового интерфейса и организации запуска приложения через промежуточную активность.
